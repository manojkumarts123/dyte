# dyte

step 1:
First node.js and npm must be installed. Then create-react-app must be installed

npm install -g create-react-app

Step 2:
create new react App using command

npx create-react-app projectreact

Step 3:
Replace the public and src folder 

Step 4:
navigate to the folder in the terminal and
Start app using following command

npm start

Step 5: App must be started
